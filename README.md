# TradeCenter — учебный проект

## Описание проекта
Система для просмотра и заказа товаров в торговых центрах.

## Структура проекта
- **DataLibrary** — библиотека с моделями, контекстом БД и сервисами.
- **TradeCenterApi** — ASP.NET Core Web API для работы с товарами
- **TradeCenterApp** — WPF приложение(главное окно, окно авторизации, окно просмотра заказовов для администратора и менеджера).
